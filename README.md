- caps_lock + jkli : => recrée un pavé directionnel

- caps_lock + delete_or_backspace => suppr   (caps_lock se comporte comme fn ici, pour éviter les erreurs de frappe et avoir 1 seul touche de modifier
- caps_lock + ru => ()
